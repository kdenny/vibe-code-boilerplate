# Grandfathered Tickets

This file tracks tickets that existed before this boilerplate was adopted. These tickets may not follow current conventions but are recognized as valid.

## Format

Each entry should follow this format:

```
## [TICKET-ID]
- **Original Title**: The original ticket title
- **Date Grandfathered**: YYYY-MM-DD
- **Reason**: Why this ticket doesn't follow conventions
- **Notes**: Any additional context
```
